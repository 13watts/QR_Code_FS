.SH NAME
magnify_pane.py \- create a local pan/zoom HTML magnifier for a 2×2 QR pane (SVG or PNG).
.SH SYNOPSIS
.B py -u magnify_pane.py
.I --in PANE.(svg|png) --out DIR
[\fB--name\fR BASE] [\fB--init-scale\fR S] [\fB--render-png\fR] [\fB--png-width\fR PX]
.SH DESCRIPTION
Copies a 2×2 pane SVG/PNG into an output directory and generates a self-contained HTML viewer that supports smooth zoom (mouse wheel, zoom buttons) and pan (click-drag). This is intended for human inspection of high-density QR mosaics.
.SH OPTIONS
.TP
.B --in PATH
Input pane file (SVG or PNG).
.TP
.B --out DIR
Output directory for the copied asset and generated viewer HTML.
.TP
.B --name BASE
Optional base name for outputs (default: input stem).
.TP
.B --init-scale S
Initial zoom scale in the viewer (default: 2.0).
.TP
.B --render-png
If input is SVG, also render a large PNG using cairosvg (optional).
.TP
.B --png-width PX
Rendered PNG width if --render-png (default: 8192).
.SH OUTPUT
Writes the following into --out:
.IP \[bu] 2
.svg or .png (copied)
.IP \[bu] 2
_viewer.html (pan/zoom viewer)
.IP \[bu] 2
(optional) _render.png (if --render-png and cairosvg available)
.SH EXAMPLES
.nf
py -u magnify_pane.py --in Z:\\slide_out\\panes\\slide_00001_pane_00000001.svg --out Z:\\slide_out\\panes_view --init-scale 4
py -u magnify_pane.py --in ./panes/pane.svg --out ./view --render-png --png-width 12000
.fi
.SH NOTES
If --render-png is used, cairosvg must be installed. On some Windows systems, Cairo runtime dependencies may also be required.
