.TH REBUILD_FROM_SLIDE_V3 1 "2026-02-21" "QRFS" "User Commands"
.SH NAME
rebuild_from_slide_v3.py \- rebuild QRFS bundles and original files from one or more slide artifacts.
.SH SYNOPSIS
.B py -u rebuild_from_slide_v3.py
.I --slides-dir DIR --out DIR
[\fB--format\fR {auto|svg|pdf|gds|oas}] [\fB--glob\fR PATTERN] [\fB--max-slides\fR N]
[\fB--decoder\fR {auto|opencv|pyzbar}] [\fB--verbose\fR]
[\fB--check-block-sha\fR] [\fB--reject-bad-blocks\fR] [\fB--emit-qr-bundles\fR]
[\fB--use-proof\fR] [\fB--proof\fR PNG] [\fB--pdf-dpi\fR N]
[\fB--quiet-modules\fR N] [\fB--force-full-gds\fR]
.SH DESCRIPTION
Reads slide artifacts and decodes QR tiles to reconstruct the original file(s). This tool is multi-slide aware and is designed to treat the slide artifact (typically SVG) as the authoritative source that would be lithographed and later scanned.
.SH FORMATS
.TP
.B svg
Recommended. Decodes tile images from SVG  payloads. If the SVG references external images, use --use-proof to crop from a proof PNG.
.TP
.B pdf
Optional. Requires PyMuPDF. Best used with a sibling SVG (for exact tile geometry) or slide JSON.
.TP
.B gds/oas
Experimental. Intended for 2×2 panes emitted by slice_slide_2x2_v2.py. Full-slide GDS/OAS decode is not implemented.
.SH OPTIONS
.TP
.B --slides-dir DIR
Directory containing slide_*.svg (recommended) and/or other supported formats.
.TP
.B --out DIR
Output directory for recovered files.
.TP
.B --format {auto|svg|pdf|gds|oas}
Input format selector (default: auto). Auto prefers SVG if present.
.TP
.B --glob PATTERN
Optional glob pattern for selecting inputs (depends on --format).
.TP
.B --max-slides N
Limit number of slides processed (testing).
.TP
.B --decoder {auto|opencv|pyzbar}
Decoder backend preference (default: auto).
.TP
.B --verbose
Verbose logging to stderr.
.TP
.B --check-block-sha
Validate embedded SHA-256 per data block.
.TP
.B --reject-bad-blocks
If a block fails embedded SHA, discard it rather than keep it.
.TP
.B --emit-qr-bundles
Also reconstruct a .qrfs-style bundle directory per file (writes decoded .qr images).
.TP
.B --use-proof
If SVG does not embed tile images, crop tiles from proof PNG.
.TP
.B --proof PATH
Proof PNG path (single slide); otherwise proof_.png is auto-detected.
.TP
.B --pdf-dpi N
PDF render DPI for tile clip extraction (default: 600).
.TP
.B --quiet-modules N
Quiet zone modules for GDS/OAS pane decoding (default: 4).
.TP
.B --force-full-gds
Allow attempting decode on non-2×2 GDS/OAS (not recommended).
.SH EXAMPLES
.nf
py -u rebuild_from_slide_v3.py --slides-dir Z:\\slide_out --out Z:\\recovered --format auto --check-block-sha --verbose
py -u rebuild_from_slide_v3.py --slides-dir ./slide_out --out ./recovered --format svg
.fi
.SH EXIT STATUS
0 if at least one file is written and verified; non-zero otherwise.
